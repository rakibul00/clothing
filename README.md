# 📦 InventoryPro - Advanced Inventory & POS Management System

InventoryPro is a full-stack, mobile-responsive Inventory Management and Point of Sale (POS) system built with **Django**, **Tailwind CSS**, and **HTMX**. It is designed specifically for small to medium-sized businesses to manage their sales, products, and reports efficiently.

## 🚀 Key Features

* **Dynamic POS Terminal**: A fast and interactive POS interface for real-time sales processing.
* **Inventory Management**: Full control over product listings, stock levels, and categorization.
* **Financial Dashboard**: Track total revenue, total sales, and returns with a clean, visual interface.
* **Responsive Sidebar**: A modern, mobile-friendly navigation sidebar that works across all devices.
* **Real-time Search**: Instant product searching using HTMX without page reloads.
* **Automated Invoicing**: Generates professional invoices immediately after completing a sale.

## 🛠️ Tech Stack

* **Backend**: Django (Python)
* **Frontend**: Tailwind CSS, HTML5, JavaScript
* **Interactivity**: HTMX
* **Database**: SQLite (Default) or PostgreSQL
* **Icons**: Font Awesome
